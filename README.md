# Wildlife Monitoring using live audio

## Run fastAPI

```
cd backend
pip install --upgrade pip
pip install -r requirements.txt
.\venv\Scripts\activate
uvircorn main:app --reload
```

FastAPI will start running on <http://localhost:8000>
